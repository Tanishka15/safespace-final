# SafeSpace Website - MVP Development Plan

## Core Files to Create/Modify:

1. **index.html** - Update title and meta tags for SafeSpace
2. **src/pages/Index.tsx** - Homepage with hero section and personas
3. **src/pages/SafetyHub.tsx** - SOS button, location sharing, safe routes
4. **src/pages/WellnessHub.tsx** - Mood tracker, breathing exercises, mindfulness
5. **src/pages/Resources.tsx** - Educational content, professional support
6. **src/pages/SecurityDirectory.tsx** - Campus security contacts by building
7. **src/pages/Profile.tsx** - User profile with preferences and contact info
8. **src/App.tsx** - Update routing for all new pages

- Navigation with SafeSpace branding
- SOS Emergency Button (prominent red button)
- Mood Tracker with emoji scale
- Breathing Exercise component

- Security contact cards
- Profile form with preferences
